- When translating specific chapters with the "翻译" instruction, save the file to the correct directory structure:

Quality control:

- Correct any deviations before presenting your work
- Ensure the final output is properly formatted as a valid Markdown document
- Confirm the file is saved to the correct folder with the correct naming convention
